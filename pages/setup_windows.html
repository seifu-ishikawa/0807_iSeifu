⚙️ 3. 事前準備

1

PowerShellの設定

管理者権限でPowerShellを起動します：

  1. スタートメニューで「PowerShell」を検索
  2. 右クリックして「管理者として実行」を選択
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
💡 ヒント: この設定により、ローカルスクリプトの実行が可能になります。

📥 4. インストール手順

1

Node.jsのインストール

Node.js LTS版をダウンロード

📦 Node.js公式サイト

インストール後、以下のコマンドで確認：

node --version npm --version
✅ 成功例: v18.0.0以上が表示されればOK
2

Gitのインストール

Git for Windowsをダウンロード

📦 Git公式サイト

インストール後の設定：

git config --global user.name "Your Name" git config --global user.email "your.email@example.com"
3

Visual Studio Code（推奨）

VS Codeをダウンロード

📦 VS Code公式サイト
💡 重要: インストール時に「PATHに追加」オプションを有効にしてください。
4

Claude CLIのインストール

npmを使用してClaude CLIをインストール：

npm install -g @anthropic-ai/claude-cli

インストール確認：

claude --version

🔑 5. Claude Codeの設定

1

APIキーの取得

  1. Anthropic Consoleにアクセス
  2. アカウントにログイン
  3. API Keysセクションから新規キーを生成
⚠️ 注意: APIキーは安全に保管し、公開しないでください。
2

環境変数の設定

PowerShellで一時設定

$env:ANTHROPIC_API_KEY = "your-api-key-here"

システム環境変数に永続設定

  1. Windowsキー + X → 「システム」
  2. 「システムの詳細設定」をクリック
  3. 「環境変数」をクリック
  4. ユーザー環境変数に新規追加：
    • 変数名: ANTHROPIC_API_KEY
    • 変数値: 取得したAPIキー
3

Claude Codeの初期化

cd C:\Users\YourName\Projects claude init

✅ 6. 動作確認

基本コマンドのテスト

# ヘルプの表示 claude --help # バージョン確認 claude --version # 簡単な質問 claude "Hello, Claude!"
✅ 成功: 各コマンドが正常に実行されれば設定完了です！

🔧 7. トラブルシューティング

よくある問題と解決方法

問題: 'claude' is not recognized

解決:

# npmのグローバルパスを確認 npm config get prefix # パスを環境変数に追加 $env:PATH += ";$(npm config get prefix)"

問題: API認証エラー

解決:

# APIキーの確認 echo $env:ANTHROPIC_API_KEY # 再設定 $env:ANTHROPIC_API_KEY = "sk-ant-..."

診断コマンド

# システム診断 claude doctor # ログの確認 Get-Content %USERPROFILE%\.claude\logs\claude.log -Tail 50
